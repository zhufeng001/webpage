收藏    我要投稿

YAML
   概述：是一种直观的能够被电脑识别的数据序列化格式
  
   在YAML里面，结构通过缩进来表示，连续的项目通过减号"-"来表示，map结构里面的key/value对用冒号":"来分隔。
   YAML也有用来描述好几行相同结构的数据的缩写语法，数组用'[]'包括起来，hash用'{}'来包括。
例子：

[python]
root: 
  urls: 
    - {name: chen,age: 22}  #短横杠加空格，冒号后面紧跟一个空格,将生成类似：urls:[ xxx,yyy] 
    - [1,2,3,4] 
  database: 
     oracle: 
         host: loacalhost    
         
         user: root 
         
         passwd: root 
         
         db: name2ip 
     mysql: 
         host: localhost 
         user: root 
         passwd: root 
         db: name2ip 
  

使用方法：

[python]
import yaml 
f = open('test.yaml') 
x = yaml.load(f) 
x就是解析得到的list或者dist，使用序列的访问方法访问即可。

ok，简单明了的yaml！
