CentOS 6.4下Eclipse安装


一、检查系统是否安装Eclipse（我安装系统时顺带装了）

1.检查是否有eclipse运行文件

 # which eclipse

  如图：

  显示已装，运行文件在/usr/bin目录下


2.找到eclipse运行文件


3.双击打开eclipse


4.选择工作空间（默认）


5.打开


关闭欢迎界面即出现eclipse平台环境


二、下载Eclipse并安装（如果没有安装Eclipse）

1.下载地址：http://www.eclipse.org/downloads/?osType=linux


  选择64位下载


2.安装

1)将下载的压缩包文件移动到 /opt目录

# sudo mv ‘文件目录’ /opt/


  同时在/opt目录下新建一个文件夹 eclipse  

 # cd /opt

 # mkdir eclipse

2)解压文件（root权限下）

 # tar Czxvf ‘文件目录’

  非root权限下：

 # sudo tar Czxvf ‘文件目录’

  解压后的文件出现在eclipse目录，进入目录


3)双击eclipse就可以启动了
